(2) 若该函数图像与 x 轴交于 M、N 两点，求线段 MN 的长度；
(3) 当 x 为何值时，y 随 x 增大而减小？

1
求解析式
由对称轴 x = 2，得 −b/(2a) = 2，即 b = −4a
代入 A(1, 2)：a − 4a + c = 2 → c = 2 + 3a
代入 B(3, 6)：9a − 12a + c = 6 → c = 6 + 3a
联立得 a = 2, b = −8, c = 8
y = 2x² − 8x + 8
2
求 MN 的长度
令 y = 0：2x² − 8x + 8 = 0 → (x − 2)² = 0
抛物线与 x 轴只有一个交点 x = 2
MN = 0（相切，只有一个交点）
3
y 随 x 增大而减小的范围
a = 2 > 0，开口向上，对称轴 x = 2
对称轴左侧单调递减
x < 2 时，y 随 x 增大而减小
