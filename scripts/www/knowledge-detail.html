2.1 二次函数的图像与性质
图文讲解

二次函数的图像与性质是初中数学的重点内容，掌握以下关键点：

1标准形式与顶点式
一般式：y = ax² + bx + c（a ≠ 0）
顶点式：y = a(x - h)² + k，其中 (h, k) 为顶点坐标

顶点坐标公式：(-b/(2a), (4ac-b²)/(4a))

对称轴：x = -b/(2a)

2开口方向
a > 0 时，开口向上，有最小值
a < 0 时，开口向下，有最大值
|a| 越大，开口越窄（图像越"瘦"）
3对称性与增减性
二次函数图像关于对称轴对称
a > 0：对称轴左侧递减，右侧递增
a < 0：对称轴左侧递增，右侧递减

关键记忆：顶点公式、对称轴公式、开口方向判断
